Labs 01 & 02

Published September 15, 2024

In Lab 1, I learned the difference between local and remote files, practiced file management in VS Code, and published my first web page. In Lab 2, I explored HTML basics, learned how to structure text with tags, and validated my HTML files using an online tool to ensure correctness.

What I Learned

  1. Understanding the importance of file structure and management in web development.
  2. Learning the proper use of HTML elements and tags for structuring a web page.
  3. How to validate an HTML file to ensure it meets web standards.

Additional Insights

I also learned about validating HTML files using the W3C Validator, which made me more comfortable because it lets me know that my HTML documents were error-free and compliant with web standards.
